## Getting Started

### Prerequisites

* Python 3.x installed
* Jupyter Notebook or JupyterLab

### Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/yourusername/IBM-Skilled-Python-Practice.git
cd IBM-Skilled-Python-Practice
```

2. Install required dependencies:

```bash
pip install jupyter numpy requests
```

3. Launch Jupyter Notebook:

```bash
jupyter notebook
```
